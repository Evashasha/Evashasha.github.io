    安装

    npm install -S webpack 装在本地（推荐）
    npm install -g webpack 装在全局 在多个项目式，不同项目版本不同时会比较麻烦

    配置文件

    webpack.config.js
    运行webpack命令时加载的配置文件，当webpack命令运行的时候，会加载该文件，根据该文件的配置内容进行执行（node语法，被node执行）

    rawloader

    加载指定文件，并以字符串的形式返回内容


    rawLoader.js

    jsonLoader.js

    file-loader

    生成文件到指定目录，并根据配置返回路径，默认：生成的新文件名称为就文件的MD5值

    babel-loader

    加载ES2015+ 代码，然后使用Babel转译为ES5
    npm -D babel-loader babel-core babel-preset-env-webpack


    babel-loader.js

    webpack执行 看编译结果

    css-loader style-loader

    css-loader：解析css文件后，使用import加载，并且返回css代码
    style-loader：将模块的导出作为样式添加到DOM中
    css.css


    cssloader.js

    vue-loader

    vueLoader.js

    插件

    在webpack预处理，打包完成后执行
    uglifyjs-webpack-plugin
    extract-text-plugin 提取分离独立的css文件 npm i -D extract-text-webpack-plugin
    commons-chunk-plugin 读取chunks之间共享的通用模块
    webpack-dev-server 模块热替换 npm i -D webpack-dev-server
